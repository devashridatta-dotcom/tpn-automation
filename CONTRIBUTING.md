# Contributing to TPN Automation

Thank you for your interest in contributing to TPN Automation!

## How to Contribute

### Reporting Issues
If you encounter a bug, have a question, or want to suggest improvements:
1. Go to the Issues tab.
2. Open a new issue with a clear description.
3. Include logs, screenshots, or sample inputs when relevant.

### Submitting Pull Requests
To propose changes:
1. Fork the repository.
2. Create a new branch describing your change.
3. Make your changes and ensure all tests pass:
   ```bash
   pytest
